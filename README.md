# WhatsApp Connection Project using Baileys

This project demonstrates how to connect to WhatsApp using the [Baileys](https://github.com/WhiskeySockets/Baileys) library, following the official guide.

## Features

- Connect to WhatsApp via QR code or pairing code
- Auto-handles connection updates (QR, disconnects, etc.)
- Persistent authentication state (for development/demo only)
